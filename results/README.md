# `results/` — canonical outputs index

This directory holds the final, canonical result tables, structured outputs, and committed appendix figures for the **cell-state reachability oracle** — a convex-cone method that asks which minimal set of CRISPRi gene knockdowns moves a primary human CD4+ T cell toward a target transcriptional state (Th1↔Th2 polarization; a CD4+ aging axis), returning a falsifiable *reachable / provably-outside-the-cone* verdict, a minimal knockdown recipe, and a Farkas/KKT activation certificate.

*61 canonical files across 11 groups.*

## Two reachability result sets (not duplicates)

The `table*.csv` and `norman_table*.csv` families look parallel but come from **different datasets and are both canonical** — neither supersedes the other:

| Family | Dataset | Axis / target | Assay |
|---|---|---|---|
| `table1_verdict` … `table5_null_summary` | primary human CD4+ T cells (Zhu et al. 2025, Marson/Pritchard, CZI VCP) | **headline** Th2→Th1 polarization | genome-scale CRISPRi Perturb-seq |
| `norman_table1_verdict` … `norman_table5_null_summary` | K562 (Norman et al. 2019, GSE133344) | CEBPA (held out) | CRISPRa (gain-of-function) |

A third generalization axis — Replogle K562/RPE1 cross-system transfer — lives in the `generality_second_dataset_*` files.

## Figure source of truth

Two committed appendix figures live **in this directory** and are indexed above: `fig_a1_sensitivity.png` and `reviewer2_trust_checks.png`.

The **repo-facing figure set** is elsewhere and is the single source of truth:

- **`docs/figures/*.png`** — 13 committed PNGs; the 11 embedded by the canonical docs are verified `.png` (no doc embeds a `.pdf`).
- **`manuscript/figures/*`** — fig1–5 / figS1–9 (png+pdf, committed) for the paper.
- **`notebooks/figures/`** — 46 gitignored, regenerable notebook outputs (18 byte-identical dupes of `docs/figures`); **not part of the submission** — regenerated on notebook run.
